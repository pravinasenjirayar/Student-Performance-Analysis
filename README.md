 Student Performance Analysis and Prediction System

 Project Overview

This project analyzes student academic performance and predicts final scores using Machine Learning techniques.

 Objectives

* Analyze student performance data.
* Identify factors affecting academic results.
* Perform Exploratory Data Analysis (EDA).
* Build and evaluate a Machine Learning model.
* Predict student final performance.

 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

 Dataset

The dataset contains student academic information such as study hours, attendance, previous marks, assignment scores, project scores, and final scores.

 Methodology

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Building
6. Model Evaluation
7. Results and Conclusion

 Machine Learning Model

* Random Forest Regressor

 Evaluation Metrics

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

 Results

The model successfully predicts student performance and identifies important factors influencing academic outcomes.

 Conclusion

The project demonstrates how Machine Learning can be used to analyze student performance and support data-driven educational decisions.
